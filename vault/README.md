# TRIBE Knowledge Vault

This Obsidian vault contains all research, architecture knowledge, brain-mapping intelligence, marketing frameworks, agent documentation, and discovered patterns for the TRIBE content intelligence system.

- [[TRIBE-v2-Architecture]] — Model architecture, fusion transformer, prediction head
- [[TRIBE-v2-Training]] — Training data, configuration, competition results

- [[Auditory-Cortex]] — Sound, music, speech processing

- [[Prefrontal-Cortex]] — Decision-making, evaluation, planning
- [[Default-Mode-Network]] — Self-referential thought, emotion, memory
- [[Fusiform-Face-Area]] — Face detection and recognition
