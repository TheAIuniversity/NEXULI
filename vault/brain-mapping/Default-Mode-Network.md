# Default Mode Network (DMN)

## What It Processes
Self-referential thought, emotion, autobiographical memory, imagination, social cognition, mind-wandering.

Key nodes:
- **Medial Prefrontal Cortex (mPFC)**: Self-referential processing. "How does this relate to me?"
- **Posterior Cingulate Cortex (PCC)**: Autobiographical memory retrieval. "This reminds me of..."
- **Angular Gyrus**: Semantic memory, conceptual knowledge (shared with [[Language-Areas]])
- **Lateral Temporal Cortex**: Social understanding, theory of mind. "What are they thinking/feeling?"
- **Hippocampal Formation**: Episodic memory, future imagination. "I can imagine myself doing that."

## TRIBE Vertex Range
Approximate fsaverage5 vertices: 8,193-11,264

## TRIBE Predictability
Pearson r: 0.15-0.25 (moderate). DMN is inherently more variable because it reflects internal, subjective experience. But consistent patterns emerge at the population level.

## Marketing Significance
- **High activation** = viewer is emotionally engaged, relating content to self
- **Low activation** = viewer is processing externally, not connecting personally
- **mPFC spike** = "This is about ME" moment (self-relevance detected)
- **PCC spike** = Memory triggered (nostalgia, recognition, familiarity)
- **Hippocampal spike** = Future visualization ("I can see myself using this")

The DMN is the **emotional resonance engine**. Content that fails to activate the DMN feels "informative but forgettable." Content that strongly activates the DMN creates lasting memories and emotional connections.

## The PFC-DMN Anti-Correlation
In normal brain function, PFC and DMN anti-correlate: when you're evaluating rationally (PFC), you're not feeling emotionally (DMN), and vice versa.

**Exceptional content breaks this pattern.** When PFC and DMN are simultaneously active, the viewer is both emotionally invested AND actively evaluating — the "I want this AND I can justify it" state. This is the strongest predictor of conversion.

## Patterns Observed
- Personal stories activate DMN 60% more than abstract concepts
- "You" and "your" pronouns spike mPFC (self-reference)
- Nostalgia cues (music, imagery from viewer's era) strongly activate PCC
- Transformation stories ("before/after") activate hippocampal formation (future self-projection)
- Testimonials activate lateral temporal cortex (social cognition: "someone like me")
- Brand storytelling that focuses on customer (not brand) activates DMN more strongly
- Emotional music without lyrics activates DMN without competing with language processing
- Humor temporarily spikes both DMN and PFC (emotional + surprise evaluation)

## Optimization Rules
- If DMN flat throughout -> content lacks emotional connection, add personal stories or "you" language
- If DMN high but PFC low -> emotionally engaging but won't convert (entertainment, not marketing)
- If DMN drops during testimonial -> testimonial feels inauthentic (social cognition detects fakeness)
- If DMN and PFC both high -> optimal conversion state, reinforce and add CTA nearby
- If DMN spikes at brand mention -> strong brand affinity (emotional connection to brand)
- Use DMN activation to "charge" emotional energy, then channel to PFC for conversion

## Emotional Triggers by DMN Sub-Region
| Trigger | Primary DMN Node | Activation Pattern |
|---------|-----------------|-------------------|
| "You" / "your" | mPFC | Immediate spike |
| Personal story | PCC + mPFC | Sustained moderate |
| Nostalgia | PCC | Strong spike |
| Before/after | Hippocampus | Gradual build |
| Testimonial | Lateral temporal | Moderate sustained |
| Community/belonging | mPFC + lateral temporal | Broad activation |
| Achievement/pride | mPFC + PCC | Strong spike |
| Fear of missing out | Hippocampus + mPFC | Spike + sustained |

## Content Strategy Implications
1. **Open with emotion** (DMN activation) to create connection
2. **Transition to rational** (PFC activation) for evaluation
3. **Re-engage emotion** (DMN) before CTA for emotional-rational convergence
4. **CTA moment**: Both DMN and PFC active = maximum conversion likelihood
